Knowing JS Scope

There are 3 ways to declair variables in JS
  1. var
    • var has a global scope if it is not declair in a function
    • if a var is declair in a function its scope is inside the function
    • Note that it is differnt from C where a variable has block scope
  2. let
    • let has block scope
    • if not in a block it is again global
  3. const (In JS const is little different)
    • const is like the const of C but totally different
    • Scope wise const is like let
